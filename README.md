# RamanSPy README

A python package for grouped spectral visualisation and analysis. Implements many capabilities commonly needed for preprocessing, quality control and visualisation of Raman spectral data.

INSTALLATION:

pip install ramanlib

PyPI: https://pypi.org/project/ramanlib/

Find documentation for the package here: https://ramanlib.readthedocs.io/en/latest/index.html

Inspired by the Raman SOP by Guo et al: Guo, S., Popp, J. & Bocklitz, T. Chemometric analysis in Raman spectroscopy from experimental design to machine learning–based modeling. Nat Protoc 16, 5426–5459 (2021). https://doi.org/10.1038/s41596-021-00620-3
